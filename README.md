## About The App

A simple Bookshop app built with Node, Express, React and MySQL, that utilizes CRUD operations.

## Technologies

I used `reactjs`, `nodejs`, `expressjs`, `mysql`, and `Vite (Build tool)`

## Setup FrontEnd

- download or clone the repository
- run `npm install`
- run `npm run dev`

## Setup BackEnd

- download or clone the repository
- run `cd backend`
- run `npm install`
- run `npm run dev`
